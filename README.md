# GirottadaCrawler

Jogo criado por três pessoas em linguagem c.
Turma: CC1TA
Criadores: Jean(Maria Emanuele), Andre Barbosa, Pietra Reis.
Histoória: Há anos que o país está sem rei. O território está um caos, os habitantes precisam de um governador já. Porém uma antiga lenda diz que apenas quem segura o lendário cetro: Girottada, pode governar as grandes terras de Girotto, que está em uma perigosa masmorra ao sul de uma pequena vila: Girottinho. O nosso herói, o personagem que você controla, se dispôs a enfrentar a perigosa masmorra e assumir o papel de rei para o país.
Controles: w/a/s/d : Mover, i : Interagir, o : Atacar
Símbolos: < : Jogador olhando para a esquerda, ^ : Jogador olhando para cima, > : Jogador olhando para a direita, v : Jogador olhando para baixo, N : Npc, * : Parede, # : Espinho, k : Caixa, O : Botão, D : Porta fechada, @ : Chave, = : Porta aberta, L : Escada, X : Monstro 1, Y : Monstro 2, Z : Boss Final
Utilizamos ia em partes do código como ajuda.
